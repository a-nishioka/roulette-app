# ルーレットアプリ

React 製のルーレットアプリです。出目を自由に設定してスピンできます。

## 機能

| 機能 | 説明 |
|------|------|
| 出目の追加 | テキスト入力 → 「＋ 追加」または Enter で追加 |
| 出目の編集 | ✎ボタン（またはラベルをクリック）でインライン編集 |
| 出目の削除 | ✕ボタンで削除 |
| 色の設定 | 16 色のプリセットから選択（追加時・編集時どちらも変更可能） |
| スピン | START ボタンでルーレットが回転し、結果をモーダルで表示 |

## 技術仕様

- **フレームワーク**: React 19 + Vite
- **スタイリング**: CSS Modules
- **永続化**: なし（データベース不使用、リロードでリセット）
- **ルーレット描画**: SVG（パス・テキスト・円要素）
- **アニメーション**: CSS `transition`（cubic-bezier イージング、4.5 秒）

## UI/UX の考慮点（Core Web Vitals）

- **LCP**: ページ構造をシンプルに保ち、画像依存なし
- **CLS**: SVG はサイズを固定した viewBox で描画。レイアウトシフトなし
- **INP**: ボタン操作は同期的に即座に反映。スピン中は編集を無効化し誤操作を防止
- **レスポンシブ**: 860px 未満でルーレットとリストが縦積みになるグリッドレイアウト

## 16 色プリセット

赤 / オレンジ / 黄色 / シアン / ミント / ブルー / パープル / ピンク /
朱色 / 深オレンジ / 黄緑 / 深い緑 / 深いブルー / ティール / 深パープル / ラベンダー
